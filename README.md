gnuplot online editor
=========

requirement
-----------

    $ yum install gnuplot
    
installation
-----------

    $ git clone https://github.com/mlwmlw/gnuplot-web-editor.git
    $ cd gnuplot-web-editor
    $ chmod 777 ./record
